# 增城区诊所位置与距离

用高德地图显示 7 家诊所位置，并支持选择两点显示直线距离。

## 使用前准备：申请高德地图 Key

1. 打开 [高德开放平台控制台](https://console.amap.com/)
2. 登录/注册后，进入 **应用管理** → **创建新应用**（可填“诊所地图”等）
3. 在该应用下 **添加 Key**：
   - 名称：如“诊所地图 Web”
   - **服务平台选择：Web 端(JS API)**
4. 复制生成的 **Key**

## 使用步骤

1. 用记事本或 VS Code 打开 `index.html`
2. 找到第 95 行左右的 `var AMAP_KEY = '请替换为你的高德地图Key';`
3. 把 `'请替换为你的高德地图Key'` 改成你申请到的 Key，保存
4. 用浏览器直接打开 `index.html`（或通过本地服务器访问）

## 功能说明

- **输入诊所名称**：在左侧 7 个输入框中填写诊所名称（可只填部分，至少 1 家）
- **标注诊所位置**：点击「标注诊所位置」，会在广州市范围内搜索并在地图上打点（优先广州 POI）
- **两点测距**：点击「选择两点测距」，然后在地图上依次点击两个点（可以是已有标记或任意位置），会显示两点直线距离（米/公里）

地图默认中心为广州市增城区，搜索范围限制在广州市内，便于找到增城及周边的诊所。
